# ThreeJS-Water v1.0
Using custom shaders and help from a YouTube tutorial with ThreeJS I made a water effect for the Solar Fusion Software website. Also, the water effect uses linear interpolation (Lerp) for responsive website design (RWD) when resizing the width of the water.
